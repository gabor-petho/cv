# CV - Gabor Petho

A bilingual (English / Hungarian) online CV built as a single-page HTML application.

## Features

- Dark / light theme toggle
- English / Hungarian language switch
- Clickable profile picture with lightbox
- Print-friendly layout
- Fully responsive design

## Usage

Open `index.html` in a browser. No build step or server required.

## Live

Hosted via GitHub Pages: [gabor-petho.github.io/cv](https://gabor-petho.github.io/cv)
